This is the Javadoc documentation of the MIDATA platform.

The controllers, that are the classes that handle the requests to the platform can be found in the {@see controllers} package and the controllers.* subpackages.

The classes containing the various types of data that is stored in the database can be found in the {@see models} package.

Request wrappers for the different sources of requests can be found in the {@see actions} package.

The query and management engine for the data records can be found in the {@see utils.access} package.

The functions for database access can be found on the {@see utils.db} package.
