Auto Import Test

In order to test the automatic import feature you need to do the following steps:

  • Your plugin needs the 'server.js' file provided by MIDATA.
  • Install your plugin from localhost into the sandbox and run your development version on your PC.
  • Using the sandbox to authorize your plugin to access the OAuth endpoint you want to import data from.
  • Have your plugin save a configuration with autoimport enabled.
  • Using a shell 'cd' into the directory where you develop your plugin and execute the command shown below.
